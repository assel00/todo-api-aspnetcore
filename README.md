# Todo API (.NET)
ASP.NET Core Web API project built with C#.

## Features
- CRUD operations
- Entity Framework Core
- PostgreSQL
- JWT Authentication
- Role-based Authorization
- Pagination
- Sorting
- Search / Filtering
- Global Exception Handling
- Logging

## Tech Stack
- ASP.NET Core
- C#
- Entity Framework Core
- PostgreSQL

## Run locally
1. Update connection string in appsettings.json
2. Run migrations
3. Start the project
